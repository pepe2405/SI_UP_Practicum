## Базови фунцкии

### strLen()
```
int strLen(const char* str)
{
	if (!str)
		return 0;

	int result = 0;
	while (*str)
	{
		result++;
		str++;
	}
	return result;
}
```
### strCopy()
```
void strCopy(const char* source, char* dest)
{
	if (!source || !dest)
		return;
	while (*source)
	{
		*dest = *source;
		dest++;
		source++;
	}
	*dest = '\0';
}
```
